# Trashcan Sensor

Dual ultrasonic sensor setup on an ESP32 to estimate how full a trashcan is.
Built using:

- ELEGOO ESP-32 Development Board (USB-C, CP2102, 3-pack)
- 2x HC-SR04 ultrasonic sensors
- 4-channel logic level shifter (KeeYees)
- 5V 2A Arkare wall adapter

## Contents

- `docs/` – overview, wiring diagram, BOM, calibration notes
- `hardware/` – schematics and enclosure files
- `firmware/` – ESPHome configuration
- `notes/` – experiments and logs
